# packet-tracer-office-network-project-1
Descrição/Description
  EN: Networking project in Cisco Packet Tracer using subnetting, device configuration, DNS and a Web Server in a multi-department office network. 
  PT-BR: Projeto de rede no Cisco Packet Tracer com subnetting, configuração de dispositivos, DNS e servidor Web em um escritório com vários departamentos.

Objetivo do projeto
  O objetivo geral do projeto é colocar em prática habilidades fundamentais em redes como:
  - Subnetting
  - Configuração de dispositivos de redes
  - Configuração manual de endereços IP
  - Testes de conectividade entre dispositivos
  - Configuração de servidor DNS
  - Planejamento e implementação de projetos

Topologia do projeto
  A imagem da topologia está disponível na pasta /imagens
  - Três departamentos em sub-redes diferentes
  - Um roteador interligando as sub-redes
  - Um switch por departamento (sub-rede)
  - Estações de trabalho (PCs) em cada departamento
  - Impressoras nos departamentos de Vendas e Financeiro
  - Servidor com serviços de DNS e HTTP

Departamentos simulados
- Administração 
- Vendas
- Financeiro

Serviços configurados
- DNS
- HTTP (página web simples)

Endereçamento IP
  O projeto utiliza o endereço de rede 192.168.40.0 com o intuito de simular o recebimento e a correta utilização de um bloco de endereços IP. 
  Sub-redes:
  - 192.168.40.0/27 -> Administração
  - 192.168.40.32/27 -> Vendas
  - 192.168.40.64/27 -> Financeiro
  Tabela de endereçamento IP disponível na pasta /imagens e no arquivo "Tabela de endereçamento IP Projeto 1 -Página1.pdf"

Testes realizados
- Comunicação entre hosts na mesma sub-rede e em sub-redes diferentes
- Comunicação com o servidor
- Acesso HTTP à página Web do servidor local
- Verficação da resolução DNS
  Capturas de tela disponíveis na pasta /imagens

Tecnologias e conceitos utilizados
- Cisco Packet Tracer
- Subnetting
- Roteamento entre sub-redes
- DNS
- HTTP
- Configuração estática de endereços IP
- Documentação de redes


ENGLISH Version


Project objective
  The general objective of the project is to put into practice fundamental networking skills, such as:
  - Subnetting
  - Configuration of network devices
  - Manual configuration of IP addresses
  - Connectivity testing between devices
  - DNS server configuration
  - Project planning and implementation

Project topology
  The topology image is available in the /imagens folder.
  - Three departments in different subnets
  - One router interconnecting the subnets
  - One switch per department (subnet)
  - Workstations (PCs) in each department
  - Printers in the Sales and Finance departments
  - Server providing DNS and HTTP services

Simulated departments
- Administration 
- Sales
- Finance

Configured services
- DNS
- HTTP (simple web page)

IP addressing
  The project uses the 192.168.40.0 network address in order to simulate the assignment and correct use of an IP address block. 
  Subnets:
  - 192.168.40.0/27 -> Administration
  - 192.168.40.32/27 -> Sales
  - 192.168.40.64/27 -> Finance
  The IP addressing table is available in the /imagens folder and in the file “Tabela de endereçamento IP Projeto 1 - Página 1.pdf”.

Tests performed
- Communication between hosts in the same subnet and in different subnets
- Communication with the server
- HTTP access to the local server web page
- Verification of DNS name resolution
  Screenshots are available in the /imagens folder.

Technologies and concepts used
- Cisco Packet Tracer
- Subnetting
- Routing between subnets
- DNS
- HTTP
- Static IP address configuration
- Network documentation
